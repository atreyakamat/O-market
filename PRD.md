# Product Requirement Document (PRD) - OCR Marketing AI Agent

## 1. Vision
To empower LinkedIn creators and marketers by automating the generation of high-quality, brand-consistent content using vision-language models.

## 2. Target Audience
- Social Media Managers
- B2B SaaS Founders
- LinkedIn Thought Leaders

## 3. Core Features
- **Brand Knowledge Base**: A persistent store for brand identity, target audience, and key messaging.
- **AI Agent Modelfile**: A customizable system prompt that defines the agent's persona (e.g., "The Viral Storyteller").
- **Image-to-Post Workflow**: Analyze uploaded images (product shots, infographics) to extract themes and context.
- **LinkedIn Draft Generator**: Create 3 unique drafts per image with relevant hashtags and CTAs.

## 4. User Flow
1. **Onboarding**: Set up Brand Facts and Agent Modelfile in the Knowledge Base.
2. **Input**: Paste an image URL or upload an asset.
3. **Generation**: AI processes image + brand context + modelfile instructions.
4. **Output**: User receives 3 drafts, ready to copy-paste.

## 5. Success Metrics
- Reduction in content creation time from 30 mins to <1 min.
- Higher engagement via brand-consistent messaging.
